## How to ?

1. `git clone`
2. `pipenv --python 3`
3. `pipenv install`
4. `flask initdb`
5.  `flask fakedata`
6.  `flask run`
7. Et go sur [127.0.0.1:5000](http://127.0.0.1:5000)

## Tests ?

La couverture est minimal mais suffisante pour découvrir les T.U et d'intégrations avec PyTest. 

Pour les lancer un simple: `pytest tests/ --disable-warnings`
